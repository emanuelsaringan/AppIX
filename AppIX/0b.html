SI Units

The International System of Units (Systeme Internationale) which is the standard base units for measurement, it uses meter, kilogram, kelvin, etc. Another system of measurement is the English System which uses miles, pound, etc.

But for generality and comfort, SI units will be used in the whole course because it is easier to manipulate for it is expressed in the powers of 10.

Some of the Base SI Units

Quantity Name Symbol Function
Length Meter m How long in terms of dimensions?
Mass Kilogram kg How heavy the object is?
Time Seconds s Measures the duration of things.
Electric Current Ampere A Flow of electricity.
Temperature Kelvin K How hot an object is.
Luminous Intensity Candela cd Intensity of light
Amount of Substance Mole mol Quantity of substance
